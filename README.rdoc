== README

MoneyCalc

<p>MoneyCalc is a simple web application for personal financial management.</p>

MoneyCalc's basic functionalities:
<ul>
	<li>Entering data about expenses and incomes</li>
	<li>Editing expenses/incomes records</li>
	<li>Adjustment of balances</li>
	<li>Accounts support</li>
</ul>
